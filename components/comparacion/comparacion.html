Punto de partida

Objetivo de aprendizaje
Reconocer cuándo el problema es predicción y cuándo es decisión secuencial.
Conocimiento previo
Distinguir agente, entorno y política.

Un modelo predictivo puede estimar el riesgo de un evento. RL recibe esa estimación como parte del estado y decide qué hacer considerando consecuencias futuras. Si cada decisión fuera independiente, una regla, un clasificador o un bandit contextual podrían ser alternativas más simples.

  • Predecir Estimar una clase, valor o nivel de riesgo.
  • Planificar Calcular decisiones cuando el modelo es conocido.
  • Aprender a decidir Mejorar una política mediante recompensas.

Comparación entre enfoques de predicción y decisión
EnfoqueCuándo resulta útilEjemplo
ReglasLa respuesta correcta es conocida y estable.Bloquear una dirección incluida en una lista.
Supervisado o anomalíasSe necesita predecir o puntuar una observación.Producir un riesgo estimado.
Bandit contextualLa acción tiene una consecuencia inmediata, sin cambiar el futuro.Elegir una revisión para un evento aislado.
Planificación o controlLa dinámica es conocida y puede resolverse directamente.Calcular una política para este modelo pequeño.
Aprendizaje por refuerzoLas acciones afectan el futuro y los valores o la política se aprenden mediante interacción.Aprender respuestas sin requerir un modelo explícito.

Como la dinámica de esta demostración es pequeña y conocida, también podría resolverse con reglas o planificación. Q-learning se utiliza para observar el proceso de aprendizaje a partir de experiencia.

Ejemplo aplicado

La detección de anomalías asigna “riesgo alto”; la política usa ese estado para elegir “bloquear y alertar”. La estimación y la decisión son tareas diferentes.
