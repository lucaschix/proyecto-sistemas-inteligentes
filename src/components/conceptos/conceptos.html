Conceptos fundamentales de RL

Punto de partida

Objetivo de aprendizaje
Relacionar los elementos de un proceso de decisión de Markov.
Conocimiento previo
Entender por qué las acciones tienen consecuencias sucesivas.

El modelo educativo combina estos conceptos; úsalos como glosario visual durante el recorrido.

  • Agente Sistema que selecciona una acción.
  • Entorno Reglas que responden a la acción.
  • Estado Información disponible para decidir.
  • Acción Decisión ejecutada por el agente.
  • Recompensa Señal inmediata entregada por el entorno.
  • Valor Q Retorno futuro estimado para una acción.
  • Política Regla que relaciona estados con decisiones.
  • Transición Estado, acción, recompensa y estado siguiente.

Dos límites importantes

La recompensa inmediata no es lo mismo que el valor Q: Q estima retorno futuro. Terminación sería un final natural; truncamiento es un corte externo que aquí ocurre después del quinto paso.

Ejemplo aplicado

“Riesgo alto” es el estado; “bloquear” es la acción; +8 es la recompensa proxy; “riesgo medio” es el siguiente estado. Q-learning usa la transición completa para actualizar el valor Q.

Lo comprendiste si… puedes clasificar correctamente estado, acción, recompensa y siguiente estado en una transición.
