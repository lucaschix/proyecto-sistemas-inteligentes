Algoritmo simplificado: Q-learning

Punto de partida

Objetivo de aprendizaje
Interpretar la actualización Q y experimentar con alpha y gamma.
Conocimiento previo
Seguir el ciclo estado → acción → recompensa → siguiente estado.

Q-learning estima el valor Q de cada acción en un estado: el retorno descontado que espera obtener al tomar esa acción y continuar con una política greedy. Es un algoritmo off-policy: puede aprender de acciones distintas de las que recomienda su política greedy.

s, a, r, s′: estado, acción, recompensa proxy y siguiente estado.

Alpha (α): cuánto incorpora cada actualización, entre 0 y 1.

Gamma (γ): cuánto pesan los valores futuros, entre 0 y 0,9 en la demostración.

Quinto paso: el segmento se trunca, pero no termina la tarea; se conserva el término de valor futuro o bootstrap.

Si existiera terminación natural: el objetivo sería solo la recompensa. Este entorno no contiene transiciones terminales.

Ejemplo numérico: 4 → 8,4

Si Q actual = 4, recompensa = 8, mejor Q futuro = 6, alpha = 0,5 y gamma = 0,8: objetivo = 8 + 0,8 × 6 = 12,8; nuevo Q = 4 + 0,5 × (12,8 − 4) = 8,4.
